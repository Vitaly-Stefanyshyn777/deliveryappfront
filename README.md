Delivery App — це веб-додаток, де користувачі можуть замовляти їжу. Проект складається з основних сторінок: сторінки магазинів магазина, улюбленого, купони, та сторінки кошика.

## ✅ Рівень виконання

- **Рівень:** Advanced (Просунутий)
- **Статус:** Всі вимоги базового, середнього та просунутого рівнів виконані.

## 🔗 Посилання

- **Репозиторій**: [GitHub Link](https://github.com/vitalikstefanisin/deliveryappfront)

## 🛠️ Технічні вимоги

### Front-End

- **Стек**: HTML, CSS, TypeScript, Next.js 15.
- **Дизайн**: Адаптивний (Responsive), реалізований за допомогою Tailwind CSS.
- **UI Framework**: Radix UI / Shadcn (опціонально).

### Back-End

- **Стек**: Node.js (без використання Firebase або подібних сервісів).
- **База даних**: Реляційна або нереляційна (не JSON-файл).
- **Хостинг**: Власний хостинг для бекенду та бази даних.

## ✨ Функціональність за рівнями

### 🟢 Base Level (Базовий рівень)

- **Shops page**: Вибір магазину та додавання товарів до кошика (дані з БД).
- **Shopping cart page**:
  - Перегляд доданих товарів, видалення або зміна кількості.
  - Форма з полями: email, телефон, адреса.
  - Валідація всіх полів перед відправкою.
  - Збереження замовлення в БД після натискання "Submit".

### 🟡 Middle Level (Середній рівень)

- **Responsive Design**: Повна підтримка десктопних, планшетних та мобільних пристроїв.
- **Product Filtering by Category**: Фільтрація товарів за категоріями (наприклад: Троянди, Тюльпани тощо).
- **Product Sorting**: Динамічне сортування за ціною (зростання/спадання) та за назвою (A-Z).
- **Shop Filtering by Rating**: Відображення магазинів на основі діапазону рейтингу (1.0 - 5.0).

### 🔴 Advanced Level (Просунутий рівень)

- **Pagination**: Завантаження товарів порціями для покращення продуктивності.
- **Reorder Previous Order**: Можливість швидко повторити замовлення з історії замовлень. Всі товари додаються до кошика зі збереженням кількості.

### 💡 Додаткові ідеї (Реалізовано)

- **Orders History**: Пошук замовлень за email та номером телефону.
- **Coupons**: Сторінка з купонами для отримання знижок у кошику.
- **Favorites**: Система улюблених товарів.

## 🛠️ Технічний стек (Детальніше)

- **Frontend**: Next.js 15, React 19, TypeScript
- **Стилізація**: Tailwind CSS
- **Управління станом**: Zustand

## 🚀 Запуск проекту

### Передумови

- Node.js 18+
- pnpm

### Встановлення залежностей

```bash
pnpm install
```

### Запуск в режимі розробки

```bash
pnpm dev
```

Додаток буде доступний за адресою: `http://localhost:3000`

### Збірка для продакшену

```bash
pnpm build
pnpm start
```

│   ├── orders/            # Сторінка замовлень
│   ├── shop/              # Сторінка магазину

│   ├── cartStore.ts       # Стор кошика

    └── index.ts

## 🔧 Налаштування API

За замовчуванням додаток використовує мок-дані для демонстрації. Для підключення реального API:

1. Відкрийте `src/lib/api.ts`
2. Змініть `USE_MOCK_DATA` на `false`
3. Переконайтеся, що бекенд запущений на `http://localhost:3300`
